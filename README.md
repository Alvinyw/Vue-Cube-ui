# open-demo

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```

1. [Cube-ui](https://didi.github.io/cube-ui/#/zh-CN/docs/introduction)

### 功能模块
1. 拍照、手机相册选图
2. 验证码
3. 按钮水波纹
4. 多语言（简单中文、繁体中文）
5. vconsole：手机端控制台
6. MD5加密
7. px2rem：移动端适配，基于 750px 设计稿
